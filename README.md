## 🤝 介绍

屏幕控制插件，支持截屏、点击、按键等

## 📦 安装

- 直接在astrbot的插件市场搜索astrbot_plugin_screenctrl，点击安装，等待完成即可

- 也可以克隆源码到插件文件夹：

```bash
# 克隆仓库到插件目录
cd /AstrBot/data/plugins
git clone https://github.com/Zhalslar/astrbot_plugin_screenctrl

# 控制台重启AstrBot
```
